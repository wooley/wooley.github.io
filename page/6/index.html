centos6.0忘记root密码解决办法

发表于 2013-01-25 | 分类于 Linux

之前虚拟机安装的Centos6忘了root密码，这下麻烦大了。放狗后，发现了改single模式修改密码的办法，可是打入passwd一点效果没有。原来是6.0和rhel 5不同了，seinunx开启的话，是没法修改密码的。

所幸找到这篇 rhel6.0及centos6.0 centos6.2 忘记root密码解决办法 照着上边第二种，关掉selinux ,用“setenforce 0”来关闭grub，就可以正常修改密码了。

南师附中在仙林开九年制分校 由仙林3所中小学整合而成

发表于 2013-01-23 | 分类于 生活点滴

南报网讯 南京师范大学附属中学仙林学校合作办学签约仪式昨天在栖霞区委党校举行。这是继3月份栖霞区与金陵中学合作创办仙林分校之后，又一所名校落户栖霞。这意味着，仙林地区的学生义务教育阶段的公办学位不是金中分校就是南师附中分校。

据了解，为更好地管理丁家庄保障房配套学校，发挥名校示范辐射作用，栖霞区人民政府、栖霞区教育局与南京市教育局、南京师范大学附属中学合作创办区属公办“南京师范大学附属中学仙林学校”。合作创办的仙林学校将对原仙林中小学和正在新建的仙林第二中小学等三所学校进行优化整合，联合创办一所九年制公办学校，学校名称为“南京师范大学附属中学仙林学校”。据介绍，原仙林小学和原仙林中学将整体合并作为小学部，占地67020 平方米，建筑面积28558 平方米，规划为10轨60个班级的规模，彻底解决原仙林小学的招生压力；新建的仙林第二中小学将作为初中部，占地50102平方米，建筑面积23290平方米，按照12轨36个班级的规模配置设备设施，将满足仙林地区居民对公办初中教育的需求。初中部2012年秋季招收初一新生。

“栖霞区人民政府将对原仙林小学和仙林中学进行全面改造。同时安排专项资金加强教师培训，出台优惠政策吸引名优骨干教师服务栖霞。”栖霞区教育局负责人表示。南京市教育局将对合作办学进行指导，跟踪并监控合作办学质量，督促学校通过有效的干部及教师培训。

据悉，南师附中仙林学校将实行管理委员会领导下的校长负责制，管理委员会主任由栖霞区分管教育的区长担任，副主任由南京师范大学附属中学校长和栖霞区教育局长担任。南师附中将派出优秀管理团队，输出优秀办学传统、教育教学管理模式，选聘优秀教师，实行小班化教学，积极进行课程、教学等改革实验，把仙林学校办出水平，办出特色。

另悉，栖霞区教育局还将在南京市四大保障房片区之一的马群地区，引进南京市第一中学优质资源，借助名校资源共同办好栖霞教育。

此为神人啊

发表于 2013-01-16 | 分类于 段子

美国电信运营商Verizon的安全审计发现，该公司的一位顶尖程序员将自己的工作外包给中国沈阳的一家软件公司。
Verizon为雇员提供了VPN允许他们在家里工作。内部的审计发现，公司明星程序员Bob的VPN登陆日志显示他固定的从中国沈阳访问公司主服务器。调查排除了恶意程序或黑客入侵的可能性。Verizon的进一步分析发现，Bob雇用了沈阳的一家软件咨询公司去做他的日常编程工作，通过FedEx提供了他的双步认证令牌，将薪水的五分之一支付给外包公司，自己则去上网冲浪，比如在Reddit上逛几小时，然后去吃饭，再上ebay血拼，更新Facebook 和LinkedIn，最后给经理发送每日工作邮件，上床睡觉。他的计划一直行之有效，在公司人力资源眼里他是最高效的程序员之一，被认为是C, C++、Perl、Java、Ruby, PHP和Python方面的专家。目前Bob已被解雇。
